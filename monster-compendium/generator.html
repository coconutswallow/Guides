Hawthorne Bestiary: Creator Studio

Use this tool to create, manage, and submit monsters for the Hawthorne Monster Compendium.

How to Use This Tool

This tool connects directly to the guild database:

  • Login: You must log in with Discord to save or submit monsters.
  • Dashboard: View your existing drafts and their status (Draft, Pending, Approved).
  • Create/Edit: Use the form to build your statblock. Changes are local until you save.
  • Save Cloud: Saves your work to the database. If the monster is already approved, this creates a new "Pending" version.
  • Submit: Once saved, click "Submit for Review" to alert the Admin team.

Backup Option: You can still download/upload Markdown files using the buttons at the bottom right.
