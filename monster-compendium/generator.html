Hawthorne Bestiary: Markdown Generator

Use this tool to make a 2024-compatible statblock for the Hawthorne Monster Compendium.
- Click [Preview] to see how your statblock will look.
- When you're happy with it, click [Download Markdown] to save the file.
- Submit the file in #dm-requests to add your monster to the Hawthorne Field Guide.
- If you need to edit it later, use [Load Markdown] to open your saved .md file and make changes.

Note: Adding heading inside text blocks (Lore, Regional Effects) works, but please use Heading 4 (####)
