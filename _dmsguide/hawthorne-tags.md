## Hathorne DM Tags

After registering as a DM on the server, you immediately gain the <span class="role-trial-dm">Trial DM</span> role. As a Trial DM, you can run a variety of adventures in the server’s persistent world with some restrictions. Access to further DM privileges on the server is available via the <span class="role-full-dm">Full DM</span> role. Either Trial DMs or Full DMs can also obtain the <b>Homebrew</b> tag, allowing the ability to create and use homebrew monsters in their adventures. Below is a snapshot of DM privileges on the server:

### DM Privileges

| Privilege | Trial DM | Full DM |
| :--- | :--- | :--- |
| Run adventures for APL 3 to 20 | ✔ | ✔ |
| Run adventures that make sweeping changes to the setting | In collaboration with other DMs and Lore | In collaboration with other DMs and Lore |
| Run official WOTC hardcover adventures | ✘ | With a request to Lore and Rules |
| Use Fair Game monsters in adventures | ✔ | ✔ |
| Use Full DM Only monsters in adventures | ✘ | ✔ |
| Use major named NPCs in adventures | ✘ | ✔ |
| Use homebrew monsters in adventures | Requires Homebrew tag | Requires Homebrew tag |
| Predetermine T0 to T2 loot | ✔ | ✔ |
| Predetermine T3 to T4 loot | ✘ | ✔ |
| Predetermine multisession loot | ✘ | ✔ |
| Drop homebrew items and rewards | With a request to Rules | With a request to Rules |

Each privilege is further detailed later in this document. For information on how to apply for either the Full DM or Homebrew tag, see Appendix A at the end of this document.
